## Run Locally

Clone the project

```bash
  git clone https://github.com/IbraheemQadi/Bookify
```

Go to the project directory

```bash
  cd Bookify
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
